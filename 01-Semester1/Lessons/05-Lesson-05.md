### 1. النظري | Theory (10 min)

**أجزاء واجهة Sibelius**

| الجزء | الوظيفة |
|-------|---------|
| **Ribbon** (الشريط العلوي) | جميع الأدوات مقسمة في تبويبات |
| **Quick Access Toolbar** | شريط الأدوات السريع (حفظ، تراجع...) |
| **Score** (منطقة النوتة) | المكان الذي تُكتب فيه النوتة |
| **Keypad** (لوحة النوتات) | اختيار قيم النوتات والرموز |
| **Transport** (شريط التشغيل) | تشغيل، إيقاف، تسجيل |
| **Mixer** (خلاط الصوت) | التحكم بمستوى الصوت لكل آلة |
| **Status Bar** (شريط الحالة) | معلومات عن الصفحة والمازورة |
| **Navigator** (المتصفح) | عرض مصغر للصفحات |
| **Properties Panel** | خصائص العنصر المختار |

**تبويبات Ribbon الرئيسية**

| التبويب | المحتوى |
|---------|---------|
| **File** | فتح، حفظ، طباعة، تصدير |
| **Home** | قص، نسخ، لصق، إضافة مازورة |
| **Note Input** | أدوات إدخال النوتة |
| **Notations** | الرموز الموسيقية (علامات، ديناميك...) |
| **Text** | النصوص والعناوين |
| **Layout** | تخطيط الصفحة والهوامش |
| **Appearance** | مظهر النوتة (Engraving Rules) |
| **Play** | التشغيل والإعدادات الصوتية |
| **Review** | مراجعة وتعليقات |

### 2. التطبيق العملي | Practice (30 min)

**تمرين 1: استكشاف الواجهة (10 دقائق)**
1. افتح Sibelius → Blank Score → (اختر أي إعدادات)
2. جرب إظهار وإخفاء الأجزاء التالية:
   - **Keypad:** F9
   - **Mixer:** F10
   - **Properties:** F8
   - **Navigator:** Ctrl+Shift+N
3. مرر الماوس على كل أيقونة في Ribbon لقراءة التلميح

**تمرين 2: استخدام Keypad المتقدم (10 دقائق)**
Keypad له 3 صفحات (Layouts):

**Layout 1 (Common):** القيم الأساسية
- الصف 1: Ronde, Blanche, Noire, Croche, Double Croche
- الصف 2: العلامات العرضية (Sharp, Flat, Natural)
- الصف 3: النقاط (Dots)، الربط (Tie)

**Layout 2 (More Notes):** 
- Tuplets (ثلاثيات، رباعيات...)
- Tremolo
- Grace Notes

**Layout 3 (Articulations):**
- Accent, Staccato, Tenuto
- Slur, Phrase Mark

**جرب التنقل بين Layouts:** اضغط Ctrl+F9 (أو استخدم الأزرار 1-2-3 في أعلى Keypad)

**تمرين 3: تخصيص Quick Access Toolbar (5 دقائق)**
1. اضغط على السهم الصغير بجانب Quick Access Toolbar
2. اختر **More Commands**
3. أضف الأوامر المفضلة (مثلاً: Save As, Export PDF, Print)
4. انقر **OK**

**تمرين 4: التنقل في النوتة (5 دقائق)**
| الاختصار | الوظيفة |
|----------|---------|
| Ctrl+Home | الذهاب لأول مازورة |
| Ctrl+End | الذهاب لآخر مازورة |
| Page Up/Down | التنقل بين الصفحات |
| Ctrl+G | الذهاب لمازورة محددة |

---

### 3. اختصارات Sibelius (5 min)

| # | الاختصار | الوظيفة | Function |
|---|----------|---------|----------|
| 1 | **F8** | إظهار/إخفاء لوحة الخصائص | Properties Panel |
| 2 | **F10** | إظهار/إخفاء الخلاط | Mixer |
| 3 | **Ctrl+F9** | التبديل بين صفحات Keypad | Switch Keypad Layout |
| 4 | **Ctrl+G** | الذهاب إلى مازورة محددة | Go to Bar |
| 5 | **Ctrl+Alt+K** | إظهار Keypad (بديل) | Show Keypad |

### 4. مراجعة وواجب | Review & HW (5 min)

**أسئلة المراجعة**
1. كم عدد التبويبات الرئيسية في Ribbon؟
2. ما اختصار إظهار Keypad؟
3. كيف تنتقل إلى مازورة رقم 25 بسرعة؟

**الواجب المنزلي**
1. اكتب تمرين رقم (———) في كتاب الصولفيج المقرر
1. افتح Sibelius وأنشئ نوتة جديدة (أي إعدادات)
2. تخصيص Quick Access Toolbar بإضافة:
   - Export PDF
   - Print
   - Undo
   - Redo
3. جرب التنقل بين جميع تبويبات Ribbon
4. أرسل لقطة شاشة لواجهة Sibelius مع تحديد 5 أجزاء رئيسية
5. احفظ باسم `Lesson5_Interface.sib`
