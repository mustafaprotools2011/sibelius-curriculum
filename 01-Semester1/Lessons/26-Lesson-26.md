---

### 1. النظري | Theory (10 min)

**أجناس الراست**
| الجنس | الدرجات | النوع |
|-------|---------|-------|
| **راست (Rast)** | C D E♭ F | جنس أساسي، شرقي |
| **نوى (Nawa)** | G A B♭ C | جنس فرعي على درجة الجواب |
| **راست على F** | F G A B♭ | جنس انتقالي |

**مسار اللحن في الراست**
```
ابدأ بـ C (قرار) ← اطلع لجنس راست ← أنتقل لجنس نوى (G) ← ارجع للقرار
```

---

### 2. التطبيق العملي | Practice (30 min)

**تمرين 1: جنس الراست والنوى (10 دقائق)**
```
جنس الراست (C-D-E♭-F):
مازورة 1: C D E♭ F | C D E♭ F |
مازورة 2: F E♭ D C | F E♭ D C |

جنس النوى (G-A-B♭-C):
مازورة 3: G A B♭ C | G A B♭ C |
مازورة 4: C B♭ A G | C B♭ A G |
```

**تمرين 2: الانتقال بين الأجناس (10 دقائق)**
```
مازورة 1 (راست): C D E♭ F | (جنس راست)
مازورة 2 (نوى): G A B♭ C | (جنس نوى)
مازورة 3 (راست): C D E♭ F | (عودة للراست)
مازورة 4 (نزول): C B♭ A G | F E♭ D C | (نزول كامل)
```

**تمرين 3: لحن راست مع وحدة (10 دقائق)**
```
مازورة 1: C. (Dotted Noire) | D (Croche) |
مازورة 2: E♭. (Dotted Noire) | F (Croche) |
مازورة 3: G (Noire) | A (Noire) |
مازورة 4: B♭ (Noire) | C (Noire) |
مازورة 5-8: كرر مع تغييرات بسيطة
```
