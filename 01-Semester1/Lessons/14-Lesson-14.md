### 1. النظري | Theory (10 min)

#### الأرباع (Intervals) في سلم العجم
| الدرجة | الفاصل | الاسم |
|--------|--------|-------|
| C → C | 0 | Unison (اتحاد) |
| C → D | 2 | Second (ثانية) |
| C → E | 4 | Third (ثالثة) |
| C → F | 5 | Fourth (رابعة) |
| C → G | 7 | Fifth (خامسة) |
| C → A | 9 | Sixth (سادسة) |
| C → B | 11 | Seventh (سابعة) |
| C → C' | 12 | Octave (أوكتاف) |

#### الثلاثيات (Triads) في سلم العجم
| الدرجة | الكورد | النوع |
|--------|--------|-------|
| I (C) | C E G | Major Triad (كبير) |
| ii (D) | D F A | Minor Triad (صغير) |
| iii (E) | E G B | Minor Triad |
| IV (F) | F A C | Major Triad |
| V (G) | G B D | Major Triad |
| vi (A) | A C E | Minor Triad |
| vii° (B) | B D F | Diminished (ناقص) |

#### رموز الكوردات (Chord Symbols)
- C = C Major
- Cm = C Minor
- C° = C Diminished
- C+ = C Augmented

### 2. التطبيق العملي | Practice (30 min)

#### تمرين 1: كتابة الأرباع (10 دقائق)
1. Sibelius → Treble Clef → 2/4
2. اكتب الأرباع من C:
```
مازورة 1: C (Noire) | D (Noire) |  (C→D = Second)
مازورة 2: C (Noire) | E (Noire) |  (C→E = Third)
مازورة 3: C (Noire) | F (Noire) |  (C→F = Fourth)
مازورة 4: C (Noire) | G (Noire) |  (C→G = Fifth)
مازورة 5: C (Noire) | A (Noire) |  (C→A = Sixth)
مازورة 6: C (Noire) | B (Noire) |  (C→B = Seventh)
مازورة 7: C (Noire) | C' (Noire) |  (C→C' = Octave)
```

#### تمرين 2: ثلاثيات العجم (10 دقائق)
1. اكتب ثلاثيات سلم العجم:
```
مازورة 1: C E G (Blanche كل نوتة) | (ثلاثي C Major - درجة I)
مازورة 2: D F A (Blanche) | (ثلاثي D Minor - درجة ii)
مازورة 3: E G B (Blanche) | (ثلاثي E Minor - درجة iii)
مازورة 4: F A C (Blanche) | (ثلاثي F Major - درجة IV)
مازورة 5: G B D (Blanche) | (ثلاثي G Major - درجة V)
مازورة 6: A C E (Blanche) | (ثلاثي A Minor - درجة vi)
مازورة 7: B D F (Blanche) | (ثلاثي B Dim - درجة vii°)
```

**لإدخال نوتات متزامنة (Chords):**
- اكتب النوتة الأولى، ثم اضغط **Shift+[نوتة]** لإضافة النوتة الثانية في نفس الوقت

#### تمرين 3: Chord Symbols (5 دقائق)
1. حدد أول ثلاثي (C E G)
2. Tab: **Text** → **Chord Symbol** (أو Ctrl+K)
3. اكتب: **C**
4. للثلاثي التالي: **Dm**
5. للثلاثي الثالث: **Em**
6. للثلاثي الرابع: **F**

#### تمرين 4: لحن مع كوردات (5 دقائق)
```
مازورة 1: | C (Noire) | E (Noire) |  (C Major - كورد C)
مازورة 2: | G (Noire) | C (Noire) |  (C Major - كورد C)
مازورة 3: | D (Noire) | F (Noire) |  (D Minor - كورد Dm)
مازورة 4: | A (Noire) | D (Noire) |  (D Minor - كورد Dm)
مازورة 5-8: كرر
```
