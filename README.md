### 📚 المحتوى

| القسم | الوصف |
|-------|-------|
| **الفصل الأول** | الدروس 1–32 — أساسيات Sibelius والمقامات والإيقاعات |
| **الفصل الثاني** | الدروس 33–64 — تقنيات متقدمة ومشاريع تخرج |

### 📥 تحميل ملفات Word

افتح المجلد [`docx_output/`](docx_output/) لتحميل 8 ملفات DOCX (4 لكل فصل، كل ملف يغطي 8 دروس).

### 🎯 لمن هذا المنهاج؟

هذا المنهاج مصمم **لجميع الطلاب** بغض النظر عن آلتهم الموسيقية:
- 🎸 عازفي العود
- 🎹 عازفي البيانو
- 🎻 عازفي الكمان
- 🥁 عازفي الإيقاع
- 🎤 المطربين
- 🎼 الملحنين والموزعين

كل درس يحتوي على تمارين عملية يمكن تطبيقها على أي آلة موسيقية.
