# اختصارات Sibelius – القائمة الكاملة
## Sibelius Shortcuts – Master List

### إدخال النوتة | Note Input
| اختصار | الوظيفة |
|--------|---------|
| N | Note Input Mode |
| 1 | Ronde (Whole Note) |
| 2 | Blanche (Half Note) |
| 3 | Noire (Quarter Note) |
| 4 | Croche (Eighth Note) |
| 5 | Double Croche (16th) |
| 6 | Triple Croche (32nd) |
| 7 | Quadruple Croche (64th) |
| 8 | Tuplet bracket |
| . (Period) | Dotted Note |
| Shift+. | Double Dotted |

### العلامات العرضية | Accidentals
| اختصار | الوظيفة |
|--------|---------|
| + | Sharp (دييز) |
| - | Flat (بيمول) |
| = | Natural (بيكار) |
| Ctrl+Shift+Q | Quarter Tone Sharp |
| Ctrl+Shift+W | Quarter Tone Flat |
