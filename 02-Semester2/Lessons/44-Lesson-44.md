### 2. النظري | Theory (10 دقائق)

#### رموز الكوردات (Chord Symbols)

رموز الكوردات هي اختصار يستخدمه الموسيقيون لمعرفة الكوردات بسرعة. تتكون من:

```
[حرف النغمة الأساسية] + [نوع الكورد] + [أي إضافات]
```

| الرمز | المعنى | مثال (C) |
|:-----:|--------|:--------:|
| **C** | كورد كبير (Major) | C-E-G |
| **Cm** | كورد صغير (Minor) | C-E♭-G |
| **Cdim** | كورد ناقص (Diminished) | C-E♭-G♭ |
| **Caug** | كورد زائد (Augmented) | C-E-G♯ |
| **C7** | كورد سابع (Dominant 7th) | C-E-G-B♭ |
| **Cmaj7** | كورد سابع كبير (Major 7th) | C-E-G-B |
| **Cm7** | كورد سابع صغير (Minor 7th) | C-E♭-G-B♭ |

#### قلب الكوردات (Chord Inversions)

عندما يكون الكورد في **الوضع الأساسي (Root Position)**، تكون النغمة الأساسية (Root) في الأسفل.

عندما نُحرّك النغمات بحيث تكون نغمة أخرى في الأسفل، نحصل على **قلب (Inversion)**.

```
C الكبير (C-E-G):
┌─────────────────┬─────────────────┬─────────────────┐
│  الوضع الأساسي   │   القالب الأول    │   القالب الثاني   │
│  (Root Position) │ (1st Inversion)  │ (2nd Inversion)  │
│                 │                 │                 │
│  G (خامسة)      │  G (خامسة)      │  C (أساسية)     │
│  E (ثالثة)       │  C (أساسية)     │  E (ثالثة)       │
│  C (أساسية) ←    │  E (ثالثة) ←     │  G (خامسة) ←     │
│    الأسفل         │    الأسفل         │    الأسفل         │
│                 │                 │                 │
│  رمز: C          │  رمز: C/E       │  رمز: C/G        │
└─────────────────┴─────────────────┴─────────────────┘
```

#### كيف تكتب رموز القلب في Sibelius

| النوع | الترتيب (من الأسفل للأعلى) | رمز Sibelius |
|:-----:|---------------------------|:------------:|
| **الوضع الأساسي** | C — E — G (Root في الأسفل) | **C** |
| **القالب الأول** | E — G — C (3rd في الأسفل) | **C/E** |
| **القالب الثاني** | G — C — E (5th في الأسفل) | **C/G** |

#### لماذا نستخدم القلوب؟

1. **لإنشاء خط بيس متحرك (Bass Line):**
   ```
   تتابع: C → C/E → F → C/G → G → C
   البيس: C → E → F → G → G → C (يتحرك!)
   ```

2. **لجعل الانتقال بين الكوردات أكثر سلاسة:**
   - بدلاً من قفزات كبيرة في البيس، نحرك نغمة أو نغمتين فقط

3. **لتحسين الصوت العام (Voice Leading):**
   - الانتقال السلس بين الكوردات يجعل الموسيقى أكثر احترافية

### 3. التطبيق العملي | Practice (30 دقيقة)

#### تمرين 1: إدخال رموز الكوردات في Sibelius (10 دقائق)

1. **افتح Sibelius → New → Piano → C Major → 4/4**
2. **أدخل لحناً بسيطاً في المسار العلوي (Right Hand):**

```
المقياس 1: C - D - E - F
المقياس 2: G - A - B - C
المقياس 3: C - B - A - G
المقياس 4: F - E - D - C
```

3. **أضف رموز الكوردات:**
   - ضع المؤشر فوق المقياس 1، النغمة الأولى
   - اضغط **Ctrl+Alt+K** (أو **Text > Chord Symbol**)
   - اكتب: **C** → Enter
   - انتقل إلى المقياس 2 النغمة الأولى → **G** → Enter
   - المقياس 3 النغمة الأولى → **F** → Enter
   - المقياس 4 النغمة الأولى → **C** → Enter

4. **استخدم **;** و **'** للرموز:**
   - خلال كتابة رمز الكورد، **;** (فاصلة منقوطة) = ♭
   - خلال كتابة رمز الكورد، **'** (فاصلة) = ♯
   - مثال: لكتابة **B♭** → اضغط **B** ثم **;** → يظهر كـ **B♭**

5. **اضغط Space للاستماع**

#### تمرين 2: كتابة القلوب في Sibelius (10 دقائق)

1. **في ملف جديد، أنشئ مسار بيس (Bass Staff):**
   - **Home > Instruments > Add or Remove > Add Piano**
   - أو أضف مساراً ثانياً: **Home > Instruments > Other > Bass**

2. **أدخل الكوردات مع القلوب:**

```
المقياس 1: C (Root Position)   — C في البيس
المقياس 2: C/E (1st Inversion) — E في البيس
المقياس 3: F (Root Position)   — F في البيس
المقياس 4: F/A (1st Inversion) — A في البيس
المقياس 5: G (Root Position)   — G في البيس
المقياس 6: G/B (1st Inversion) — B في البيس
المقياس 7: C (Root Position)   — C في البيس
المقياس 8: C — استراحة
```

3. **خطوات محددة لكتابة C/E:**
   - اكتب النغمة E في البيس (الطبقة السفلى)
   - ضع المؤشر فوق المقياس 1 في المسار العلوي
   - اضغط **Ctrl+Alt+K**
   - اكتب: **C** ثم /** (شرطة مائلة)** ثم **E** → Enter
   - النتيجة: رمز "C/E" يظهر فوق النوتة

4. **أضف رموز الكوردات فوق المسار العلوي:**
   - المقياس 1: **C**
   - المقياس 2: **C/E**
   - المقياس 3: **F**
   - المقياس 4: **F/A**
   - المقياس 5: **G**
   - المقياس 6: **G/B**
   - المقياس 7: **C**
   - المقياس 8: (استراحة)

#### تمرين 3: تتابع هارموني مع القلوب (10 دقائق)

**تحدٍّ:** اكتب تتابعاً هارمونياً كاملاً مع القلوب

```
تتابع: C → G/B → Am → Am/G → F → C/E → Dm → G7 → C
البيس:  C →  B →  A →   G →  F →   E →  D →  F →  C
```

**في Sibelius:**
1. مسار البيس (سفلي): اكتب البيس فقط: C - B - A - G - F - E - D - F - C
2. مسار الأكورد (علوي): اكتب الكوردات الثلاثية كـ Block Chords
3. أضف رموز الكوردات: C, G/B, Am, Am/G, F, C/E, Dm, G7, C

**لاحظ:** البيس يتحرك بشكل تدريجي تنازلي: C → B → A → G → F → E → D → F → C
هذا يسمى **"Bass Line Descending"** — أحد أجمل التتابعات الهارمونية!

### 5. مراجعة وواجب | Review & HW (5 دقائق)

**أسئلة المراجعة**
1. ماذا يعني الرمز "C/E"؟ وكيف تكتبه في Sibelius؟
2. اذكر الأنواع الثلاثة لقلب الكوردات (Inversions) للكورد C Major.
3. لماذا نستخدم القلوب في التوزيع الموسيقي؟
4. ما الفرق بين "C7" و "Cmaj7"؟ (بحث صغير)
5. كيف تكتب رمز كورد B♭؟ (أي مفتاح تضغط في Sibelius؟)

**الواجب المنزلي**
1. اكتب تمرين رقم (44) في كتاب الصولفيج المقرر — اكتب 6 كوردات في كل وضع من أوضاع القلب الثلاثة (18 كورداً إجمالاً)
2. في Sibelius: أنشئ ملفاً من 8 مقاييس مع:
   - 4 مقاييس كوردات في الوضع الأساسي (Root Position)
   - 4 مقاييس نفس الكوردات مع القلوب (Inversions)
   - رموز الكوردات فوق كل مقياس
3. على آلتك الموسيقية: اعزف تتابع C → Am → F → G (I-vi-IV-V) مع محاولة جعل البيس يتحرك بسلاسة
4. ابحث عن أغنية واحدة تحتوي على كوردات مقلوبة (مثلاً أغنية "Let It Be" — فيها C/G)

### 6. تطبيق عملي | Practical Application

- **على آلتك الموسيقية:**
  - **البيانو:** اعزف C الكبير بثلاث طرق:
    1. Root Position: C-E-G (C في الأسفل)
    2. 1st Inversion: E-G-C (E في الأسفل)
    3. 2nd Inversion: G-C-E (G في الأسفل)
  - **العود:** جرب كورد C في الوضع الأساسي (C-3-5 على الأوتار)، ثم اقلبه
  - **الجيتار:** استخدم شكل C الكبير المفتوح (Root Position)، وشكل C/E (1st Inversion)
  - **أي آلة:** اعزف النغمات كـ Arpeggio في كل وضع

- **في Sibelius:**
  - أنشئ تتابعاً هارمونياً من 4 مقاييس باستخدام القلوب فقط
  - أضف رموز الكوردات فوق المسار العلوي
  - استخدم **File > Export > PDF** لتصدير النوتة كمرجع

- **نشاط إضافي:**
  - اكتب تتابع C → Dm → Em → F → G → Am → Bdim → C
  - لكل كورد، استخدم القالب الأول (1st Inversion)
  - اسمع: البيس يتحرك: C → D → E → F → G → A → B → C (سلم كامل!)

### 9. ملاحظات المدرس | Teacher Notes

- **درس القلوب قد يكون صعباً لبعض الطلاب** — استخدم البيانو الحقيقي أو لوحة المفاتيح لتوضيح الفرق
- **للتبسيط:** اعتبر القلوب مثل إعادة ترتيب غرفة — نفس الأثاث (النغمات)، لكن في أماكن مختلفة
- **نشاط يدوي:** اكتب نغمات C-E-G على 3 بطاقات، واطلب من الطالب إعادة ترتيبها بحيث تكون نغمة مختلفة في الأسفل
- **ربط مع العود العربي:**
  - في العود، القلوب أقل شيوعاً لأن العود يعتمد على الأوتار المفتوحة
  - لكن فهم القلوب يساعد في التوزيع الموسيقي الحديث وفي Sibelius
- **للطلاب المتقدمين:**
  - قدم مفهوم Slash Chords المتقدمة (مثل Dm7/G = G9sus4)
  - تحداهم بكتابة تتابع من 16 مقياساً مع 8 قلوب مختلفة على الأقل
- **تذكير:** الدرس القادم سنتعلم كيفية اختيار الكوردات الصحيحة لأي لحن — وهو تطبيق عملي لكل ما تعلمناه
