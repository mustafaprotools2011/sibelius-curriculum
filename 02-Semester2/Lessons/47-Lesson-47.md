# الدرس 47 | Lesson 47
## تناغم الألحان بناءً على المقامات العربية | Harmonizing Melodies Based on Arabic Maqamat

---

### 1. معلومات الدرس | Lesson Information

- **رقم الدرس:** 47
- **عنوان عربي:** تناغم الألحان بناءً على المقامات العربية
- **English Title:** Harmonizing Melodies Based on Arabic Maqamat
- **الوحدة:** 6 — الكوردات والهارموني | Chords and Harmonies
- **الأهداف | Objectives:**
  - فهم التحديات الخاصة بتناغم الألحان العربية (ربع التونات)
  - بناء كوردات تناسب المقامات العربية: الراست، البياتي، الحجاز
  - تطبيق أنماط مرافقة عربية (Drone, Pedal Point, Ostinato)
  - تناغم لحن عربي قصير في Sibelius مع ربع التونات

- **الإيقاع | Rhythm:** 4/4 — متنوع

### 2. النظري | Theory (10 دقائق)

#### التحدي: الهارموني الغربي والمقامات العربية

الموسيقى الغربية تعتمد على **الكوردات الثلاثية** (Triads) المبنية على نظام **النصف تون (Semitone)**. الموسيقى العربية تعتمد على **ربع التونات (Quarter-tones)**، وهذا يخلق تحدياً:

```
المشكلة:
  كورد C الكبير الغربي: C(دو) - E(مي) - G(صول)  ← كلها أنصاف تون
  مقام الراست على C:    C(دو) - E♭(مي نصف بيمول) - G(صول)  ← ربع تون!

الحل: ليس كل كورد غربي يناسب المقامات العربية
```

#### ثلاث مقاربات لتناغم الموسيقى العربية

**المقاربة 1: Drone (الدون) — النغمة المستمرة**
```
┌─────────────────────────────────────────────┐
│  أبسط وأكثر طريقة أصالة في الموسيقى العربية   │
│                                             │
│  ● اعزف نغمة واحدة مستمرة (مثل C أو D)       │
│  ● فوقها، اعزف اللحن بحرية                     │
│  ● مثال: مقام راست على C → Drone على C       │
│                                             │
│  في Sibelius: نغمة طويلة في المسار السفلي 🎵 │
└─────────────────────────────────────────────┘
```

**المقاربة 2: الكوردات المفتوحة (Open Chords / Power Chords)**
```
┌─────────────────────────────────────────────┐
│  كوردات من نغمتين فقط — تتجنب الثالثة (3rd)   │
│  وبالتالي تتجنب مشكلة ربع التون!              │
│                                             │
│  ● Root + 5th (مثل C + G)                   │
│  ● Root + 4th (مثل C + F)                   │
│  ● Root + Octave (مثل C + C أعلى)           │
│                                             │
│  مثال: مقام الراست → كوردات C5, F5, G5      │
└─────────────────────────────────────────────┘
```

**المقاربة 3: Pedal Point (نقطة الدواسة) — مقاربة متقدمة**
```
┌─────────────────────────────────────────────┐
│  نغمة واحدة مستمرة في البيس (Pedal)           │
│  بينما الكوردات تتغير فوقها                    │
│                                             │
│  مثال: في مقام راست:                          │
│  البيس: C مستمر لمدة 4 مقاييس                 │
│  الكوردات: C, F/C, G/C, C (كلها فوق C)       │
└─────────────────────────────────────────────┘
```

#### كوردات تناسب المقامات العربية الرئيسية

**مقام الراست (Rast — على C):**
| الدرجة | النغمات | كورد مقترح | ملاحظة |
|:------:|---------|:----------:|--------|
| C (1) | C-D-E♭-F-G-A-B-C' | **C5 (C+G)** | تجنب الثالثة (ربع تون) |
| D (2) | D-E♭-F♯-G-A-B-C-D' | **Dm** أو D5 | D-F-A (ثالثة صغيرة) |
| F (4) | F-G-A-B♭-C-D-E♭-F' | **F5 (F+C)** أو F | جرب F مع C فقط |
| G (5) | G-A-B♭-C-D-E♭-F♯-G' | **G5 (G+D)** | تجنب B♭ - D (ثالثة صغيرة) |

**مقام البياتي (Bayati — على D):**
| الدرجة | النغمات | كورد مقترح |
|:------:|---------|:----------:|
| D (1) | D-E♭-F-G-A-B♭-C-D' | **Dm** (D-F-A) |
| G (4) | G-A-B♭-C-D-E♭-F-G' | **Gm** أو **G5** |
| A (5) | A-B♭-C-D-E♭-F-G-A' | **Am** أو **A5** |

**مقام الحجاز (Hijaz — على D):**
| الدرجة | النغمات | كورد مقترح |
|:------:|---------|:----------:|
| D (1) | D-E♭-F♯-G-A-B♭-C-D' | **Dm** أو **D(#4)** |
| G (4) | G-A-B♭-C-D-E♭-F♯-G' | **Gm** |
| A (5) | A-B♭-C♯-D-E♭-F♯-G-A' | **A7** (A-C♯-E-G) |

### 3. التطبيق العملي | Practice (30 دقيقة)

#### تمرين 1: تناغم لحن بمقام الراست (10 دقائق)

1. **افتح Sibelius → New → Piano → 4/4 → 8 مقاييس**

2. **اضبط العلامات العرضية للمقام:**
   - مقام الراست على C: E نصف بيمول (Quarter-tone Flat)
   - استخدم **Ctrl+Shift+W** لوضع نصف بيمول على E
   - أنشئ المفتاح الموسيقي: اختر Custom Key Signature
   - أو ببساطة: اكتب E مع نصف بيمول يدوياً

3. **أدخل اللحن الراستي البسيط:**

```
المقياس 1: C - D - E♭ - F
المقياس 2: G - A - B♭ - C'
المقياس 3: C' - B♭ - A - G
المقياس 4: F - E♭ - D - C
(كرر أو ابتكر لحنك الخاص)
```

**ملاحظة:** E♭ هنا تعني E نصف بيمول (Quarter-tone flat E)

4. **أضف المرافقة — مقاربة Drone:**
   - في المسار السفلي، اكتب C3 طويلة (نوتة كلها) — استمرار Drone
   - أو: استخدم نمط C5 (C-G-...-G) متكرر

5. **أضف رموز الكوردات:** 
   - كل مقياس → **C5** (رمز كورد C مع خامسة فقط)
   - أو استخدم **C** فقط (Sibelius سيعزف C الكبير، لكننا نستخدم Drone منفصل)

6. **اضغط Space للاستماع** — هل تسمع الطابع الراستي؟ 🎵

#### تمرين 2: تناغم لحن بمقام البياتي (10 دقائق)

1. **ملف جديد → Piano → 4/4 → 8 مقاييس**
2. **العلامات العرضية للبياتي على D:**
   - E♭ (نصف بيمول على E) — Ctrl+Shift+W
   - B♭ (بيمول عادي على B) — اضغط B ثم -

3. **لحن بياتي بسيط:**

```
المقياس 1: D - E♭ - F - G
المقياس 2: A - B♭ - C - D'
المقياس 3: D' - C - B♭ - A
المقياس 4: G - F - E♭ - D
(مقاييس 5-8: كرر أو ابتكر)
```

4. **المرافقة — مقاربة Open Chords + Pedal:**
   - المسار السفلي: D2 نغمة مستمرة (Pedal Point)
   - أضف كوردات مفتوحة: D5 (D-A), G5 (G-D), A5 (A-E)
   - أو: استخدم Dm (D-F-A) — F يناسب مقام البياتي

5. **رموز الكوردات:**
   - لكل مقياس: **Dm**, **Gm** أو **D5**, **A5**
   - ملاحظة: رمز D5 يكتب كـ **D(no3)** في Sibelius (كورد بدون ثالثة)

#### تمرين 3: تناغم لحن بمقام الحجاز (10 دقائق)

1. **ملف جديد → Piano → 4/4 → 8 مقاييس**
2. **العلامات العرضية للحجاز على D:**
   - E♭ (نصف بيمول) — Ctrl+Shift+W
   - F♯ (دييز عادي) — F ثم +
   - B♭ (بيمول عادي) — B ثم -
   - (اختياري) C♯ — C ثم +

3. **لحن حجازي مميز:**

```
المقياس 1: D - E♭ - F♯ - G
المقياس 2: A - B♭ - C♯ - D'
المقياس 3: D' - C♯ - B♭ - A
المقياس 4: G - F♯ - E♭ - D
المقاييس 5-8: ابتكر لحنك الخاص
```

4. **المرافقة — مميزة للحجاز:**
   - المسار السفلي: Dm (D-F-A) — لكن F♯؟ هنا التحدي!
   - **الحل:** استخدم Drone على D + كوردات بدون ثالثة
   - أو: استخدم A7 (A-C♯-E-G) في الأماكن المناسبة

5. **رموز الكوردات المقترحة:**
   - المقاييس 1-2: **Dm** (أو **D5**)
   - المقياس 3: **A7** (عند النزول)
   - المقياس 4: **Dm** (العودة إلى D)

6. **استمع إلى الصوت الشرقي الجميل!** 🎶

4. ما الفرق بين Power Chord (C5) و Major Chord (C)؟

**الواجب المنزلي**
1. اكتب تمرين رقم (47) في كتاب الصولفيج المقرر — دوّن سلم الراست (C) والبياتي (D) مع ربع التونات

   - ملف 1: لحن راستي مع مرافقة Drone

3. على آلتك الموسيقية: اعزف لحناً بياتياً بسيطاً مع Drone على D (نغمة مستمرة)
4. استمع إلى مقطوعة عربية تقليدية (أم كلثوم، فريد الأطرش، أو عبد الوهاب) وحاول تحديد: هل هناك هارموني؟ أم تعتمد على Drone فقط؟

### 6. تطبيق عملي | Practical Application

- **على آلتك الموسيقية:**
  - **العود:** اعزف لحناً على مقام الراست (C). استخدم الوتر C كـ Drone (اتركة مفتوحاً) طوال اللحن
  - **الناي / الكمان:** اعزف اللحن فقط مع مستمع يتخيل Drone (أو استخدم تطبيق Drone على الهاتف)
  - **البيانو / الكيبورد:** اعزف Drone بيد يسرى (C مستمرة) + لحن راستي بيد يمنى
  - **أي آلة:** اعزف اللحن + تخيل Drone في ذهنك

- **في Sibelius:**
  - استخدم **Ctrl+Shift+W** لتجربة نصف البيمول على نغمات مختلفة
  - استمع إلى الفرق بين E العادي و E نصف بيمول — كم هو جميل!
  - أضف علامات أداء (Dynamics) لجعل الأداء أكثر تعبيراً

- **نشاط إضافي — "تحدي التوزيع العربي الحديث":**
  - خذ لحناً حجازياً (8 مقاييس)
  - حاول توزيعه بثلاث طرق:
    1. Drone (تقليدي عربي)
    2. Power Chords (حديث غربي-عربي)
    3. كوردات غربية كاملة (كوزمبوليتان)
  - أي توزيع تفضل؟ ماذا يضيف كل توزيع إلى اللحن؟

### 9. ملاحظات المدرس | Teacher Notes

- **هذا الدرس هو جسر بين الهارموني الغربي والموسيقى العربية** — قد يكون محورياً للطلاب الذين يهتمون بالموسيقى العربية
- **للطلاب الذين لم يدرسوا المقامات من قبل:** راجع معهم أساسيات المقامات (الراست، البياتي، الحجاز) قبل البدء
- **نقطة مهمة:** لا توجد إجابة واحدة صحيحة لتناغم الموسيقى العربية — كل مقاربة تعطي نتيجة مختلفة
- **للطلاب المتقدمين:**
  - قدم مقاربة "Bichord" (كوردات من نغمتين فقط مع إيقاع معقد)
  - تحداهم بكتابة توزيع كامل لمقام صبا (Saba) — أصعب لكنه جميل جداً
- **ربط مع الواقع:**
  - الموسيقى العربية المعاصرة (أغاني المهرجانات، البوب العربي) تستخدم الكوردات الغربية بكثرة
  - الموسيقى العربية التقليدية (القدود، الموشحات) تعتمد على Drone و Ostinato
  - كلا الأسلوبين صحيح — الفرق هو السياق الموسيقي
- **نشاط سمعي:** اعزف نفس اللحن الراستي مع 3 مقاربات مختلفة واطلب من الطلاب التصويت على المفضلة لديهم
- **للطلاب المهتمين بالتأليف:** هذا الدرس هو بداية رحلة التوزيع الموسيقي العربي الحديث
- **تذكير:** الدرس القادم هو مشروع الكوردات والهارموني — ختام الوحدة 6!
