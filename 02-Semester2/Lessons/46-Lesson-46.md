### 2. النظري | Theory (10 دقائق)

#### ما هو نمط المرافقة (Accompaniment Pattern)؟

نمط المرافقة هو **طريقة محددة لعزف الكوردات** بشكل متكرر. بدلاً من عزف الكورد كـ Block Chord في كل مقياس، نستخدم أنماطاً إيقاعية متنوعة تجعل الموسيقى أكثر حيوية.

```
الكورد (C Major) يمكن عزفه بعدة طرق:
┌──────────────────────────────────────┐
│  Block Chord:  ┌───┐                 │
│                │C E│                 │
│                │ G │                 │
│                └───┘                 │
├──────────────────────────────────────┤
│  Alberti Bass: C - G - E - G         │
│               (منخفض-مرتفع-وسط-مرتفع) │
├──────────────────────────────────────┤
│  Arpeggio:     C - E - G - C (صاعد)  │
│  Broken Chord: C - G - E - G (متنوع) │
├──────────────────────────────────────┤
│  Rhythmic:     C C/E/G - - C C/E/G   │
│               (كورد على النبضات القوية)│
└──────────────────────────────────────┘
```

#### الأنماط الأساسية

**1. Alberti Bass (ألبرتي باس) — كلاسيكي:**

```
نمط: منخفض - مرتفع - وسط - مرتفع
لكل كورد C:
C3 - G3 - E3 - G3
 ↑    ↑    ↑    ↑
(1)  (2)  (3)  (4)
```
- أكثر نمط استخداماً في الموسيقى الكلاسيكية (موسيقى موزارت وهايدن)
- يعطي إحساساً بالحركة والأناقة

**2. Block Chords على النبضات — بوب / روك:**

```
نمط: كورد كامل على النبضات القوية فقط
C(كامل) - - - - C(كامل) - - - -
   ↑               ↑
  نبضة 1          نبضة 3
```
- يناسب الموسيقى الحديثة والبوب
- بسيط لكنه فعّال

**3. Arpeggio صاعد / نازل — رومانسي:**

```
نمط صاعد: C - E - G - C (الأوكتاف الأعلى)
نمط نازل: C - G - E - C (العودة)
نمط دائري: C - E - G - E (صعود + نزول سريع)
```
- يناسب الموسيقى الرومانسية والبطيئة
- يعطي إحساساً بانسيابية

**4. نمط مزودج (Doubled) — بوب معاصر:**

```
اليد اليسرى: C - G (خامسة فقط - Power Chord)
اليد اليمنى: C - E - G (Block Chord عالٍ)
```
- شائع في موسيقى البوب والروك
- يعطي صوتاً قوياً دون تعقيد

### 3. التطبيق العملي | Practice (30 دقيقة)

#### تمرين 1: نمط Alberti Bass في Sibelius (10 دقائق)

1. **افتح Sibelius → New → Piano → C Major → 4/4 → 8 مقاييس**

2. **في المسار السفلي (Left Hand)، اكتب نمط Alberti:**

```
المقياس 1 (C Major): C3 - G3 - E3 - G3
المقياس 2 (G Major): G2 - D3 - B2 - D3
المقياس 3 (Am):      A2 - E3 - C3 - E3
المقياس 4 (F Major): F2 - C3 - A2 - C3
المقياس 5-8: كرر نفس النمط
```

3. **كيفية الإدخال بسرعة:**
   - ادخل النغمات الأربعة لكل مقياس يدوياً
   - استخدم **R** (تكرار) لتكرار النمط في المقاييس التالية
   - أو: اكتب المقياس الأول، اختره، ثم **Ctrl+C** و **Ctrl+V** في المقاييس الأخرى
   - غيّر النغمات الأساسية فقط لكل كورد

4. **في المسار العلوي (Right Hand)، أضف لحناً بسيطاً:**
   - اختر نغمات من سلم C الكبير
   - استخدم أثمان مختلفة (♩ و ♪)

5. **اضغط Space للاستماع** — النمط الكلاسيكي الجميل! 🎶

#### تمرين 2: نمط Arpeggio صاعد-نازل (10 دقائق)

1. **في ملف جديد، استخدم الميزان 3/4 (الڤالس):**
   - **Ctrl+Shift+T → 3/4**

2. **في المسار السفلي، اكتب نمط Arpeggio لكل كورد:**

```
المقياس 1 (C):        C3 - E3 - G3   |  النمط: صاعد (1-2-3)
المقياس 2 (F):        F2 - A2 - C3   |  النمط: صاعد
المقياس 3 (G7):       G2 - B2 - D3 - F3 | 4 نغمات (كورد سابع)
المقياس 4 (C):        C3 - G3 - E3 - C3 | نازل (عالٍ → منخفض)
```

3. **لنمط الڤالس الكلاسيكي (Oom-pa-pa):**
   - النبضة 1: Root (منخفضة) — "Oom" 🥁
   - النبضة 2: كورد كامل (في الوسط) — "pa" 🎹
   - النبضة 3: كورد كامل (في الوسط) — "pa" 🎹

```
C المقاييس 1-2 (3/4):
نبضة 1: C2 (منخفض جداً) 
نبضة 2: C3-E3-G3 (كورد في الوسط)
نبضة 3: C3-E3-G3 (كورد في الوسط)
```

4. **جرب هذا النمط مع تتابع C - Am - Dm - G7:**

```
المقياس 1 (C):  C2 → (C3-E3-G3) → (C3-E3-G3)
المقياس 2 (Am): A2 → (A2-C3-E3) → (A2-C3-E3)
المقياس 3 (Dm): D2 → (D2-F3-A3) → (D2-F3-A3)
المقياس 4 (G7): G2 → (G2-B2-D3-F3) → (G2-B2-D3-F3)
```

#### تمرين 3: نمط البوب الإيقاعي الحديث (10 دقائق)

1. **ملف جديد → Piano → C Major → 4/4 → 8 مقاييس**

2. **النمط الحديث "Eighth Note Pulse" (نبض الثمن):**

```
المقياس 1 (C): C3-E3-G3 (على 1) + C3-E3-G3 (على 3) 
المقياس 2 (G): G2-B2-D3 (على 1) + G2-B2-D3 (على 3) 
المقياس 3 (Am): A2-C3-E3 (على 1) + A2-C3-E3 (على 3)
المقياس 4 (F): F2-A2-C3 (على 1) + F2-A2-C3 (على 3)
```

3. **أضف تعقيداً — نمط Syncopated (متزامن):**
   - اكتب الكوردات على النبضات المتأخرة (Off-beats):

```
المقياس 1: استراحة(1) + C3-E3-G3(2+) + استراحة(3) + C3-E3-G3(4+)
المقياس 2: استراحة(1) + G2-B2-D3(2+) + استراحة(3) + G2-B2-D3(4+)
```

4. **لإدخال الاستراحات:** استخدم **0** (صفر) على Keypad مع النوتة المرادة

5. **في المسار العلوي، أضف لحناً متناغماً مع الإيقاع**

6. **Space للاستماع** — هذا يبدو كأغنية بوب حقيقية! 🎵

### 5. مراجعة وواجب | Review & HW (5 دقائق)

**أسئلة المراجعة**
1. اذكر 3 أنماط مرافقة (Accompaniment Patterns) مختلفة.
2. ما هو نمط Alberti Bass؟ وكيف يعزف؟
3. كيف يختلف نمط "Oom-pa-pa" عن "Eighth Note Pulse"؟
4. لماذا نستخدم أنماط مرافقة مختلفة بدلاً من Block Chords فقط؟

**الواجب المنزلي**
1. اكتب تمرين رقم (46) في كتاب الصولفيج المقرر — دوّن 4 أنماط مرافقة مختلفة في 4 مقاييس (مقياس واحد لكل نمط)
2. في Sibelius: أنشئ ملفاً من 16 مقياساً مع:
   - لحن بسيط في المسار العلوي
   - أنماط مرافقة مختلفة في المسار السفلي:
     - المقاييس 1-4: Alberti Bass
     - المقاييس 5-8: Arpeggio صاعد
     - المقاييس 9-12: Waltz (Oom-pa-pa)
     - المقاييس 13-16: Eighth Note Pulse
   - رموز الكوردات فوق اللحن
3. على آلتك الموسيقية: اعزف نفس اللحن مع 3 أنماط مرافقة مختلفة — أي نمط تفضل ولماذا؟
4. استمع إلى أغنية من اختيارك وحاول تحديد نمط المرافقة المستخدم

### 6. تطبيق عملي | Practical Application

- **على آلتك الموسيقية:**
  - **البيانو:** اعزف تتابع C-G-Am-F بأربعة أنماط مختلفة:
    1. Block Chords (كورد على كل نبضة 1)
    2. Alberti Bass (منخفض-مرتفع-وسط-مرتفع)
    3. Waltz (Oom-pa-pa)
    4. Arpeggio صاعد
  - **العود:** جرب الكوردات كـ "Rolled Chords" (أربيجيو دائري)
  - **الجيتار:** استخدم نمط "Strumming" (الريشة على الأوتار) — وهو نمط مرافقة بحد ذاته
  - **أي آلة:** اختر نمطاً يناسب آلتك واعزف به

- **في Sibelius:**
  - استخدم **Home > Arrange > Copy** لنسخ النمط بسرعة
  - استخدم **Edit > Filter > Select All** ثم **Ctrl+D** لتكرار الأنماط
  - أضف Dynamics (f, p, crescendo) لتحسين الأداء

- **نشاط إضافي — "تغيير نمط المرافقة يغير المزاج":**
  - خذ تتابع C-G-Am-F
  - طبّق عليه 3 أنماط مرافقة مختلفة
  - سجّل ملاحظاتك: كيف تغير المزاج؟
