## الإيقاعات الخليجية – الخبيتي والبندري
## Khaliji Rhythms – Al-Khibiti and Al-Bandari

---

### 1. النظري | Theory (10 min)

#### الخبيتي (Al-Khibiti) – 2/4
- إيقاع سريع، حماسي
- نمط: **DUM - - TAK TAK** (أو DUM TAK DUM TAK مع اختلافات)
- أصل: السعودية - الخليج

| DUM | (Rest) | TAK | TAK |
| Noire | (Noire Rest) | Noire | Noire |

#### البندري (Al-Bandari) – 4/4
- إيقاع شعبي خليجي
- نمط: **DUM TAK DUM TAK TAK TAK**
- يُستخدم في الرقصات الشعبية

#### مميزات الإيقاعات الخليجية
- إيقاعات حية، سريعة
- استخدام التصفيق (Hand Clap) كجزء من الإيقاع
- دقات قوية ومنتظمة

---

### 2. التطبيق العملي | Practice (30 min)

#### تمرين 1: الخبيتي (10 دقائق)
1. Sibelius → Treble Clef → 2/4 → C Major
2. اكتب إيقاع الخبيتي:
```
مازورة 1: C (Noire) | (Rest Noire) |  (DUM -)
مازورة 2: C (Croche) C (Croche) | C (Croche) C (Croche) |  (TAK TAK TAK TAK)
```
**النمط المبسط:**
```
| DUM | - | TAK | TAK |
| C (Noire) | (Whole Rest في 2/4 = Noire Rest) | C (Croche) C (Croche) | C (Croche) C (Croche) |
```

#### تمرين 2: البندري (10 دقائق)
1. غير إلى 4/4
2. اكتب إيقاع البندري:
```
مازورة 1: C (Noire) | C (Noire) | C (Noire) | C (Noire) |  (DUM DUM DUM DUM)
```
**النمط الكامل:**
```
| DUM (يمين) | TAK (يسار) | DUM (يمين) | TAK (يسار) TAK (يمين) TAK (يسار) |
مازورة: C (Noire) | C (Croche)(Croche Rest) | C (Noire) | C (Croche)C(Croche)C(Croche) |
```

#### تمرين 3: لحن عجم مع خبيتي (10 دقائق)
```
مازورة 1: C (Noire) | E (Noire) |  (خبيتي)
مازورة 2: G (Noire) | C (Noire) |
مازورة 3: E (Noire) | F (Noire) |
مازورة 4: G (Ronde) |
مازورة 5-8: كرر مع بندري
```

### 5. تطبيق العود | Oud Application
اعزف إيقاع الخبيتي على العود: ريشة قوية (DUM) - سكتة - ريشة خفيفة (TAK) - ريشة خفيفة (TAK). كرر.
